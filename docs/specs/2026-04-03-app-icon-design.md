# 程序图标设计说明

日期：2026-04-03

## 目标

为 `ImageFormatConverter.exe` 添加一个简洁、易识别的应用图标，使其在桌面和资源管理器中更像正式的小工具软件。

## 已确认方向

- 图标风格：图片卡片 + 转换箭头
- 视觉倾向：亮色、高辨识度、偏小软件图标
- 图标内容：纯图形，不带文字

## 设计要点

- 以一张简化的图片卡片作为主体，表达“图片文件”
- 右下角叠加一个圆形转换箭头，表达“格式转换”
- 主体色使用偏亮的蓝色系，辅以少量暖色高光，增强桌面识别度
- 保持图形简洁，避免在小尺寸下因为细节过多而发糊

## 交付范围

- 生成 Windows 可用的 `.ico` 图标文件
- 更新打包脚本，使最终 `ImageFormatConverter.exe` 使用该图标
- 重新打包并验证生成结果

## 非目标

- 本轮不制作整套品牌视觉系统
- 本轮不做多套图标主题切换
- 本轮不引入文字缩写元素
